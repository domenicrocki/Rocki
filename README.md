# TETRIS // NEON FUTURE

Ein komplettes Tetris-Spiel mit futuristischen Neon-Effekten – direkt im Browser spielbar.

## Features

- **Volle Tetris-Mechanik**: SRS-Rotation mit Wall Kicks, 7-Bag-Randomizer, Hold-Piece, 3er-Next-Queue, Ghost Piece, Hard Drop, DAS (Delayed Auto Shift)
- **Scoring**: Punkte pro Level skaliert, Combo-System, Tetris-Bonus (4 Zeilen)
- **Levels**: Alle 10 Zeilen steigt das Level, Geschwindigkeit nimmt zu

## Futuristische Effekte

- **Glitch-Text-Animation** auf Titel & Overlays
- **Neon-Glow-Bloecke** mit Circuit-Pattern und Pulsieren
- **Partikel-System**: Hintergrund-Netzwerk, Explosionen bei Line Clears, Tetris-Burst, Level-Up-Ring, Drop-Trail
- **Scanline-Overlay** (CRT-Effekt)
- **Screen-Flash** bei Line Clears
- **Combo-Popups** mit Skalier-Animation
- **Level-Up-Flash** am Board-Rand

## Audio (Web Audio API, kein externer Sound noetig)

- **Korobeiniki-Melodie** (Tetris-Theme) synthetisiert
- Sound-Effekte: Move, Rotate, Drop, Hard Drop, Line Clear, Tetris, Combo, Level Up, Game Over
- Mute-Toggle mit **M**

## Steuerung

| Taste | Aktion |
|-------|--------|
| ← → | Bewegen |
| ↓ | Soft Drop |
| ↑ | Drehen |
| Space | Hard Drop |
| C | Hold |
| P/Esc | Pause |
| M | Mute |

Plus **Touch-Support** (Swipe & Tap) fuer mobile Geraete.

## Starten

Einfach `index.html` im Browser oeffnen und loslegen!
